EXACT DATA PERSISTENCE REQUIREMENTS
=====================================

CRITICAL: Use identical localStorage keys and data structures:

localStorage Keys (MUST BE EXACT):
- "ticketapp_session" - session token
- "ticketapp_user" - current user object
- "ticketapp_users" - array of all registered users
- "ticketapp_tickets" - array of all tickets

User Object Structure:
{
  id: string,
  email: string,
  name: string
}

Ticket Object Structure:
{
  id: string,
  title: string,
  description?: string,
  status: "open" | "in_progress" | "closed",
  priority?: "low" | "medium" | "high",
  createdAt: string (ISO),
  updatedAt: string (ISO)
}

Session Token Format:
"token_" + timestamp + "_" + random

ID Generation Format:
User ID: "user_" + timestamp
Ticket ID: "ticket_" + timestamp + "_" + random

=====================================
STATE MANAGEMENT (PINIA STORES)
=====================================

AUTH STORE (stores/auth.ts):
Required state:
- user: User | null
- isAuthenticated: boolean

Required actions:
- login(email: string, password: string): Promise<boolean>
- signup(email: string, password: string, name: string): Promise<boolean>
- logout(): void
- initializeAuth(): void (check localStorage on app start)

Login Logic (EXACT):
1. Get users from localStorage "ticketapp_users"
2. Find user with matching email and password
3. If found: generate token, store user and token, return true
4. If not found: return false

Signup Logic (EXACT):
1. Check if user already exists with email
2. If exists: return false
3. If not: create user, add to users array, auto-login, return true

TICKETS STORE (stores/tickets.ts):
Required state:
- tickets: Ticket[]

Required actions:
- addTicket(ticketData: Omit<Ticket, "id" | "createdAt" | "updatedAt">): void
- updateTicket(id: string, updates: Partial<Ticket>): void
- deleteTicket(id: string): void
- getTicketStats(): { total: number, open: number, inProgress: number, closed: number }
- loadTickets(): void
- saveTickets(): void

ROUTING REQUIREMENTS

EXACT Routes (Vue Router):
- "/" - LandingPage.vue
